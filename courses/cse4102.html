1D+2D Initial Setup:
Windows:
Follow Video tutorials step by step:
1. Install GCC compiler:
How to Install MinGW (GCC/G++) Compiler in Windows 10
Add this Path as well as:
C:\MinGW\msys\1.0\bin
To check whether flex and bison are installed, open command line and write:
flex
bison
[If command is not recognized, goto that directory and check whether flex and bison are in there. If not, then search for flex and bison in main directory of MinGW and add path containing flex and bison in Path variable]
2. Install Visual Studio Code:
How to install Visual Studio Code on Windows
3. Install Visual Studio plugin to run code easily:
C++ with Visual Studio Code. (Easiest setup)

Ubuntu:
1. GCC compiler is usually installed in Ubuntu by default. To check whether gcc is installed:
which gcc
If not installed:
sudo apt-get install gcc
2. Install flex and bison:
sudo apt-get install flex
sudo apt-get install bison
To check whether flex and bison are installed:
which flex
which bison
3. Install Visual Studio Code on Ubuntu Linux:
Visual Studio Code on Ubuntu
4. Install Visual Studio plugin to run code easily:
C++ with Visual Studio Code. (Easiest setup)

Mac:
MacOS users have to install gcc, visual studio (+extensions), flex and bison. They can take help from youtube to install all these libraries.

Install all these applications and libraries before attending the sessional class.
